About
I'm a full stack developer with a background in accounting. I've built a porfolio of projects to showcase my skills. My main project is a tool that can be used to inform investment decisions. Users can assign importance to financial ratios and rank investments based on how well they score.
Projects
Movie Master
Users can search movies based on keywords and filters. Information about each movie can be viewd upon clicking the movie. Users can create accounts and save their favourite movies. This application was built using a react front end and a node.js and express backend. Authentication is handled by JWT using passport middleware.

Character Finder
As with the game Where's Waldo, users have to search an image full of many familiar characters to find a short list of characters. The time it takes users to complete the game is tracked and users have the option of entering their name afterwords to post themselves to the leaderboard. The front of this application was built using react. It fetches from a back end I created using Node.js and express.
